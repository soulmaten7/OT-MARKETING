2. 시안별 상세 스펙 (제작자용)

각 시안의 플레이스홀더 위치·글자 수 제한·톤 가이드 원문은 landings/_templates/CONTENT_GUIDE.md 파일 참조. 이 파일은 샘플 치환 작업 시 보면서 진행.

🅐 신뢰·법률 (Navy + Gold)

  • Hero 헤드라인: 15~25자 · 존대형 · 숫자/경력 근거
  • 강조 어구: 8~15자 · 골드 색상 적용
  • 혜택 3카드: 제목 10~18자, 설명 60~100자
  • 어울리는 업종: 개인회생, 주식리딩, 부동산, 법률·세무

🅑 액션·전환 (Orange + Yellow)

  • Hero 헤드라인: 15~25자 · 혜택/할인 직접 표기
  • 노란 형광펜: 10~20자 · 가격 또는 즉시 혜택
  • Stats 4숫자: 반드시 4개 필수
  • 어울리는 업종: 정수기렌탈, 인터넷통신

🅒 케어·친근 (Mint + Cream)

  • 톤: ~해요 친근 존대 · 재촉·단정 금지
  • 심리적 안전 단어: 편하게, 부담 없이, 함께
  • 어울리는 업종: 병의원, 개인회생 감성판

🅓 미니멀·에디토리얼 (Black/White + Red)

  • 톤: 매거진 · 사실 중심 서술 · 감탄 최소
  • 어울리는 업종: 부동산분양, B2B 컨설팅, 프리미엄 서비스

🅔 다크·테크 (Dark + Cyan)

  • 필수 고지: "수익률은 과거 실적, 미래 보장 X" 금융권 고지 필수
  • 어울리는 업종: 주식리딩, 핀테크, IT SaaS

🅕 비비드·모바일 (Purple + Coral + Lime)

  • 톤: 젊고 경쾌 · SNS 광고 느낌 · 이모지 활용
  • 어울리는 업종: 정수기렌탈, 인터넷통신, 소비재 모바일

3. 업종별 법적 고지 raw 템플릿

공통 (모든 업종 적용)

본 사이트는 상담 안내를 목적으로 합니다.
제공 내용은 상담 후 개인 상황에 맞게 안내됩니다.
개인정보보호법·표시광고법 준수.

개인회생·법률

구체 법률 판단은 제휴 변호사의 상담을 통해 제공됩니다.
변호사법 준수. 상담이 법률대리를 보장하지는 않습니다.

의료

본 광고는 의료광고 사전심의를 받았습니다 (심의번호 {번호}).
의료상담은 면허 보유 의료인이 담당합니다.
시술 효과·부작용은 개인차가 있을 수 있습니다.

금융·투자

본 사이트는 특정 상품 가입을 권유하지 않습니다.
수익률은 과거 실적이며 미래를 보장하지 않습니다.
투자 원금 손실 가능성이 있습니다.

통신·렌탈

제공 혜택은 가입 조건에 따라 변경될 수 있으며,
최종 제공 여부는 상담 후 확정됩니다.
약정 기간·위약금 조건은 계약서 기준.

부동산

본 사이트는 부동산 중개가 아닌 상담 안내 목적입니다.
분양 조건·가격·시점은 분양사 공식 발표 기준.
청약 가점 산정은 본인 확인 필요.

4. 제작자 체크리스트 (광고주 시안 치환 시)

  • 광고주로부터 회사 기본 정보 수령 (회사명·대표자·사업자번호·주소·전화·이메일)
  • 광고주 차별화 포인트 3가지 수령 (제목+설명)
  • 절차 4단계 수령
  • 업종별 법적 고지 확인 (위 섹션 3 참조)
  • 과장 표현 체크 (무조건, 100%, 반드시 등 금지)
  • 근거 없는 수치 체크 (실측 자료 확보)
  • 이미지 저작권 확인 (상업용 라이선스 여부)
  • 시안 확정 후 Next.js 랜딩으로 변환 · Vercel 배포
  • 폼 제출 → 광고주 수신 경로 테스트 (구글 시트·이메일)

5. 업종 확장 방법

신규 업종 (예: "탈모", "보험", "학원") 추가 시:

  1. docs/industry-kits/0X-new-industry/ 폴더 생성
  2. 광고주_가이드.md, 랜딩_카피.md, 광고_이미지_스펙.md, 법적_고지.md 4개 파일 작성 (템플릿은 docs/industry-kits/01-debt-relief/ 참조)
  3. public/samples.html 의 industryData 객체에 신규 업종 키 추가
  4. industrySelect 드롭다운에 <option value="new"> 추가 (disabled 해제)
  5. PDF 변환: pandoc 광고주_가이드.md -o 광고주_가이드.pdf
